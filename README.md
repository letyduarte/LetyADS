# LetyADS - Portfolio de Marketing e Tráfego Pago

Site moderno e profissional para portfólio de especialista em marketing digital e tráfego pago para clínicas de estética e odontologia.

```

├── styles.css          # Todos os estilos CSS
├── script.js           # JavaScript interativo
```

## 🚀 Como Personalizar

### Editar Cores
Abra `styles.css` e altere as variáveis no `:root`:

```css
:root {
    --primary: #sua-cor-aqui;
    --secondary: #sua-cor-aqui;
    /* ... etc */
}
```

### Editar Conteúdo
Edite `index.html` para:
- Trocar nome/contatos em Header
- Atualizar descrições de serviços
- Adicionar seus próprios casos
- Modificar formulário de contato
- Atualizar redes sociais

### Adicionar Funcionalidades
Edite `script.js` para:
- Backend para formulário (actualmente é simulado)
- Dark mode toggle
- Menu mobile
- Mais animações
- Analytics

## 🌐 Deployment

Para colocar online:

1. Fazer upload de todos os 3 arquivos (HTML, CSS, JS) para seu hosting
2. Manter a mesma pasta/estrutura
3. Configurar domínio/HTTPS
4. Testar em diferentes devices

**Sugestões de Hosting:**
- Vercel (recomendado para estático)
- Netlify
- GitHub Pages
- AWS S3
- Qualquer hosting com suporte a HTML/CSS/JS
