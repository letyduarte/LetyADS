## 🚀 Como Usar

### 1. **Abrir no Navegador**
```
1. Salve os 3 arquivos na mesma pasta
2. Clique duplo em index.html
3. Voilà! Site aberto
```

### 2. **Fazer Edições**
```
1. Abra em editor de texto (VS Code, Notepad++)
2. Edite o arquivo desejado
3. Salve (Ctrl+S)
4. Recarregue navegador (F5)
```

### 3. **Fazer Upload**
```
1. Faça upload dos 3 arquivos para seu hosting
2. Mande o link para www.seu-site.com
3. Pronto! Site ao vivo
```
